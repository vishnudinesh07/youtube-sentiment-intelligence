### 📌 Project Overview

YouTube generates thousands of comments every day, but reading and interpreting them manually is inefficient.

**YouTube Audience Intelligence** was built to solve that problem.

This platform allows users to paste a public YouTube video link and instantly receive:

✅ Sentiment breakdown of viewer comments  
✅ Audience mood insights  
✅ Positive / Neutral / Negative trends  
✅ Multilingual feedback analysis  
✅ Executive summary dashboard  
✅ Comment-level intelligence  

---

### ✨ Core Features

#### 🎯 Audience Sentiment Dashboard

- Overall sentiment score  
- Positive / Neutral / Negative distribution  
- Viewer reaction summary  
- Comment volume analysed  

#### 🌍 Multilingual Intelligence

Supports audience comments across multiple languages.

- English comments → VADER Sentiment Engine  
- Non-English comments → Hugging Face NLP API  

#### 📊 Interactive Analytics

- KPI metric cards  
- Progress indicators  
- Pie chart sentiment split  
- Styled comment intelligence table  
- Executive insights panel  

#### ⚡ Production Architecture

- Lightweight backend API  
- Cloud hosted frontend  
- Optimised memory usage  
- Stable public deployment
